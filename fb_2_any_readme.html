FB2Any Readme

FB2Any is a set of tools, allowing you to convert FictionBook 2.0 documents to various formats. This installer is provided by Dmitry Gribov, with any comments and/or suggestions feel free to email author.

Before submiting bug reports, install latest version.

This program is distributed in the hope that it will be useful, but without any warranty; without even the implied warranty of merchantability or fitness for a particular purpose.

FB2Any is built as a set of COM objects and may be utilized from any application/script. This installer contains sample scripts with most common functionality, but you can go further and edit those scripts to accomplish more dedicated tasks. If it's not enough - contact author.

It is recommended to install FictionBook tools and Any2FB in addition to FB2Any to get a basic set of FB2 tools.

Also examine "MoreXSL" folder to find alternate XSL's for your format.

For more information about FB2 standard, fb2-related applications and other useful stuff go to FictionBook homepage.
