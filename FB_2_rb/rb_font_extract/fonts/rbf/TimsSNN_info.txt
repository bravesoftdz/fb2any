This is a replacement for NuvoMedia's Rocket eBook default fonts.
The package contains Russian 12pt small, 16pt large Times New Roman
fonts and modified fixed-width Courier New font.

Note that all fonts are Russian, not Cyrillic (i.e. characters in range
0xC0-0xFF and 0xA8/0xB8 "yo" are replaced with Russian ones, but no other
Cyrillic letters are included).
